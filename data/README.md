Data access and size
The raw datasets exceed GitHub size limits, so they are not included in this repo.
All data sources are public.
File paths in the notebook point to local copies.

Data sources
• EPA Air Quality System, county level AQI and PM2.5 data
• U.S. Energy Information Administration, electricity and natural gas consumption
• State and federal transportation reports, ZEV and LDV sales
• Public fuel price datasets
